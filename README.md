# 项目名称：基于 MindSpore 的自动驾驶真实雨景视觉任务

## 项目目的

1. 学习 MindSpore 框架
2. 实践去雨算法 [算法讲解](https://zhuanlan.zhihu.com/p/344422879)
3. 部署 yolect++实现语义分割 [算法讲解](https://blog.csdn.net/yx868yx/article/details/113779675)

## 项目结构

├─.vscode  
├─data 存放数据集  
├─utils  
├─mindspore  
├─efficientderain 去雨算法  
├─yolact 语义分割  
├─main.py  
└─README.md
